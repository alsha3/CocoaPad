The Source Code

Since CocoaPad is open source, the source code is given away. The code, along with some other stuff, is located in the /Extras/Source/ folder on your installer disk image. The actual Xcode project files and Objective-C source code are in the /Extras/Source/Source Code/ folder.
The project files require Mac OS X 10.3 Panther and Xcode Tools 1.0 or later (they are .xcode files). The source files (.h and .m files) can be viewed with any text editor.

Be sure to read the README in the Source/ folder, so you know which files are which, how to build, etc.

Although CocoaPad is relatively simple, you might want to understand the code better. That's why we've included the Developer Notes file, which contains notes on the program, the files, etc. Developer Notes is located in /Extras/Source/.

If you have problems, you can always contact me at betacheese@gmail.com.

Have fun!
