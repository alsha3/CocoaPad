Opening a file

Okay, now you're back in CocoaPad, and you want to open a document you saved earlier. No problem. Just like saving, opening a document in CocoaPad is like opening a document in any other application.

To open a file:

  1. Pull down the File menu, and select Open (or press Command-O).
  2. An open dialog box will appear. Choose where you want to save your document from the popup menu.
  3. Finally, click the Open button, and your document appears in it's own window.

If you want to use a Finder-like display for navigating to a different folder, click the blue down arrow.
You can also use the Open button on the toolbar. If the Open button is not on the toolbar, then pull down the Window menu, and select Customize Toolbar…. Then, drag the Open item from the customization sheet to the toolbar.
