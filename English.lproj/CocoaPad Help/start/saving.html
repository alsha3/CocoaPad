Saving a file

Once you've finished typing up all of that, you would want to preserve it, wouldn't you? It's easy to do so. Saving a document in CocoaPad is like saving a document in any other application.

To save a file:

  1. Pull down the File menu, and select Save (or press Command-S).
  2. A save sheet will appear. Don't worry about the file format. For now, choose where you want to save your document from the popup menu.
  3. Finally, click the Save button, and the deed is done.

If you want to use a Finder window-like display for navigating to a different folder, click the blue down arrow. Also, you can use Save As… to save your file somewhere else.
You can also use the Save and Save As… buttons on the toolbar. If the Save As… button is not on the toolbar, then pull down the Window menu, and select Customize Toolbar…. Then, drag the Save As… item from the customization sheet to the toolbar.
