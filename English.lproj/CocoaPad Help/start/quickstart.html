QuickStart

Here's a quick rundown of the typical process of creating, editing, and saving a document.

  1. Double-click CocoaPad's icon in your Applications folder to launch it. A new blank document should appear. If you've already launched CocoaPad, pull down the File menu, and select New (or press Command-N).
  2. Type up your document. It could be a letter, a memo, or anything.
  3. Save your document by pulling down the File menu, and selecting Save (or press Command-S).
  4. Make sure you want to save your document in the folder name that's shown. Click the Save button to save your document.
