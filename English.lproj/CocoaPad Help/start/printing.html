Printing a document

Printing is as simple as ever for any Mac OS X program that uses Cocoa technology. CocoaPad is no exception.
You need a printer hooked up if you plan to print anything. (Duh.)

To print a document:

  1. Pull down the File menu, and select Print (or press Command-P).
  2. A print sheet will appear. Here, you can set how much of your document you want to print out, the layout of the document, and many other settings. For more information, click the button with the question mark in the bottom-left corner.
  3. Now, click the Print button, and your document will start to print.

Once again, for information about printing in Mac OS X, click the button with the question mark in the bottom-left corner of the print sheet.
If you want to configure additional settings for your printout, such as page orientation (i.e. the direction the document will appear on the paper, for example, horizontally or vertically), then pull down the File menu, and select Page Setup (or press Command-Shift-P). For more information, click the button with the question mark in the bottom-left corner.
Unlike many other computer systems (such as Microsoft Windows and the old Macintosh), there is no seperate "Print Preview" menu item. Instead, a button labeled Preview will appear in the print sheet. Click this to view a preview of how your document will look when it is printed out.
