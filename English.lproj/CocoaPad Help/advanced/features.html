Suggesting features

If you want to suggest a feature that's missing, then you can contact the developer of CocoaPad (me) by dropping me an e-mail at betacheese@gmail.com. Tell me what feature you want, where it would be (e.g. Format menu), and maybe how to implement it (if you're a programmer).

Otherwise, you're going to have to live without this feature for a while.
If you're a programmer who wants to see the source code, please see The Source Code topic.
