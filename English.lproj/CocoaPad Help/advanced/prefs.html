Fiddling with preferences

You can wreak such havoc as changing the document's default background color, saying if a new document opens on startup, and more!

To edit the preferences:

  • Select Preferences… from the the CocoaPad menu (or press Command-,), or click the Preferences button on the toolbar (if it's visible).
  • Select the tab that would contain the settings you want to edit.

Because the preference panel is a big place, here's a list of all the tabs:

  • Settings: Set settings that dealing with a document or other operation of CocoaPad.
  • Fonts: Set the default fonts used in CocoaPad documents.
  • Colors: Set the default colors used in CocoaPad documents.
  • New Document Format: Set the document format CocoaPad uses when it creates a new document.
  • Automatic Backup: Turn Automatic Backup on or off, and specify how often CocoaPad should backup documents. (More on Automatic Backup)

Resetting the preferences to their original (when you first installed CocoaPad) state is as simple as clicking Reset to original settings and clicking OK. But you can't undo this.
Tip: If you want to find out what a certain preference means, hold the mouse over the preference for a while. A Help Tag will popup with an explanation of what the preference will do.
Tip: If you want certain settings to be applied right now to all currently open documents (the window says which ones will be applied), click the Apply To All… button in the Settings tab.
