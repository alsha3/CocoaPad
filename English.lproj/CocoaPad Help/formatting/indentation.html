Paragraph indentation

You can set how much a paragraph is indented by moving a tab stop on the ruler.

  1. Look for a horizontal bar on the ruler that looks like this: .
    This may not be clear at first, since the indentation tab stop initially sits atop of the first page margin. So, if you don't see the horizontal bar on the ruler at first, then look for something like this: . The top part of this is the indentation tab stop, which you can move around.
  2. Move it to how far you want paragraphs to be indented.

If you don't select any text to be indented, all text in between the first and last Return in the paragraph/line is indented.
