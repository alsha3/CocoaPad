Changing your text font

After a while, you're gonna get bored of that same old font for everything. Wouldn't it be fun to change the font to something else?

To change the font:

  1. Click the Fonts button on the toolbar, or pull down the Font menu, and select Show Fonts (or press Command-T).
  2. The font panel will appear. Select the font you want in the Font Family column, the style in the Typeface column, and the size in the Size column.
  3. Finally, click the close button to dismiss the font panel.

Tip: Fonts can be applied immediately to selected text.
