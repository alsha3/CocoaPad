Stylesheets

You're operating system version needs to be at least Mac OS X 10.3 (Panther) in order to use stylesheets. To purchase Mac OS X 10.3, or to learn more about it, visit Apple's Mac OS X website.

A stylesheet is a collection of "styles", which is like a font with preset attributes. An example of a style could be blue, shadowed, centered, bold 24-point Helvetica, and it could be named "Big Header". This is useful if you need preset fonts and settings for different types of text, like captions, headlines, etc.

To make a new style:

  1. Apply desired formatting to a piece of text.
  2. Choose Other… from the Styles popup menu on the ruler.

    If you want to jump to another piece of formatted text (perhaps you liked the formatting of the next paragraph better), click the arrow buttons ( and ) to "scroll" through the styles. The arrow buttons with the lines ( and ) jump to the first and last pieces of formatted text, and the square button () returns you to the original piece of formatted text you had.

  3. Click the Add To Favorites button, and type a name for your new style. You'd probably want to check the Include the font as part of the style checkbox, which includes the font and color information in the style. Checking the Include the ruler as part of the style will include ruler settings, such as tab stops and justification.
  4. Finally, click Add. Your style will be available in all Mac OS X programs that use the standard Mac OS X ruler and use the stylesheet system, including CocoaPad.

To use a style:

  1. Select the piece of text you want to format.
  2. Choose the desired style from the Styles popup menu on the ruler.

If you want to remove a style you added, choose Other… from the Styles popup menu on the ruler, click the Favorite Styles radio button, and then select the style you want to remove from the popup menu. Click the Remove From Favorites button to delete the style (be warned: you cannot undo this).
