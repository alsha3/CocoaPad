If you want MORE typography stuff…

CocoaPad may be simple, but it's a powerful word processor. It supports such advanced typography features such as kerning, ligatures, writing direction, etc., previously only found in big, expensive page layout programs.

If you want to use these features, explore the Font and Text menus. However, we'll quickly cover what you can do:

  • Kern (in the Font menu): Used to alter the amount of space between letters. Use the Tighten command to scrunch letters together, the Loosen command to widen them apart, and the Use Default or Use None command to return them to normal.
  • Ligatures (in the Font menu): Ligatures are fancy characters made from usually two letters (such as fl and fi) that have parts of their characters joined with the other one. Choosing Use Default or Use All will substitute, for example, fi, with the appropriate ligature. Choosing Use None will get rid of ligatures. Not all fonts, such as Helvetica, have ligatures, but a lot, such as New York, Charcoal, and Hoefler Text, support ligatures.
  • Baseline (in the Font menu): The baseline controls how far text dips below or rises up from the norm. Choose Raise to raise text bit by bit above normal level, and Lower to lower text bit by bit below normal level. The Baseline menu also comes with two handy commands, Superscript and Subscript, which are essentially predefined baseline levels for your text. You probably already know what these do. Superscript raises text above the line, which can be useful in math equations, like (x + y)2 = x2 + 2xy + y2. Subscript lowers text below the line, and can be useful in chemical formulas (for example, CO2).
  • Character Shape (in the Font menu): Some fonts offer special ways of making characters—the sole item in this submenu toggles the character shape between Traditional Form and the font's specialized character shapes. According to Apple, "This feature is currently used to specify traditional shapes in Chinese and Japanese scripts, but font developers could use it for other scripts as well."
  • Copy Style and Paste Style (in the Font menu): Used to copy and paste text with formatting intact.
  • Writing Direction (in the Text menu): Select Right to Left to toggle right to left writing, where text types from the right to the left.
