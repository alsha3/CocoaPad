Adding images

If you want to add some illustrations, by all means, go ahead! It's extremely easy to add graphics.
Make sure that your document is in CocoaPad format or Rich Text Format with Attachments before continuing. For more information, see All about file formats.

To add a graphic:

  1. Copy a graphic from another application.
  2. Paste it into your document.

Or use "drag-and-drop":

  • Drag an image file from the Desktop/Finder into your document.

If something is already selected when you use drag-and-drop, the image will replace it. You can undo this, though, but you'll have to drag the image back into your document again.
