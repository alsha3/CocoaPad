Spell checking

CocoaPad has a built-in spell checker. It's easy to use, but it isn't so smart!

To spell check your document:

  • Choose Spelling or Check Spelling from the Edit > Spelling menu.

Tip: You can have CocoaPad automatically note a misspelling by choosing Check Spelling As You Type from the Edit > Spelling menu. However, this is only temporary, and after you close the document, you have to choose Check Spelling As You Type again. To make it so that documents by default check spelling as you type, check the Check Spelling As You Type checkbox in Preferences. For more on preferences, see Fiddling with preferences.
