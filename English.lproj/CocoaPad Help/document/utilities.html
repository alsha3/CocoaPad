"Utilities"

CocoaPad has some handy little features that can capitalize your text, or convert your document to RTF format (see All about file formats). Here's a rundown of these miscellaneous tools:

  • CAPITALIZE Selected Text: Capitalizes the selected text (or all of the text if nothing is selected).
  • Make Selected Text lowercase: Makes the selected text (or all of the text if nothing is selected) to lowercase.
  • Capitalize First Letter Of Each Word: Capitalizes the first letter of each word in the selected text (or all of the text if nothing is selected).
  • Word Count: Shows you how many words, paragraphs, and characters there are currently in your document.
  • Change Background Color: Changes the background color of your document. This is only temporary though—it will not be saved. This utility cannot be undone once it is used.
  • Convert to CocoaPad format: Enables your document to be saved in CocoaPad format (see All about file formats). The conversion cannot be undone once it is used. However, you can always convert your document back to the original format.
  • Convert to Rich Text Format: Enables your document to be saved in Rich Text Format (see All about file formats). The conversion cannot be undone once it is used. However, you can always convert your document back to the original format (although you won't get your graphics back).
  • Convert to Rich Text Format with Attachments: Enables your document to be saved in Rich Text Format with Attachments (see All about file formats). The conversion cannot be undone once it is used. However, you can always convert your document back to the original format.
  • Convert to Microsoft Word Format: Enables your document to be saved in Microsoft Word format, yes, Word format (see All about file formats). The conversion cannot be undone once it is used. However, you can always convert your document back to the original format (although you won't get your graphics back).
  • Convert to Plain text format: Enables your document to be saved in Plain text format (see All about file formats). The conversion cannot be undone once it is used. However, you can always convert your document back to the original format (although you won't get your graphics and formatting back).

These are all found in the Format menu.
If these sound an awful lot like "Tools" in other programs, then you're right. Utilities are really Tools, except we like to call them "Utilities".
All utilties can be undone, unless specified otherwise.
