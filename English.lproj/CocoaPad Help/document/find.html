Finding (and replacing) text

If you have a big document and want to find something in it, or if you want to replace something that you typed throughout your document, use the Find panel.
You're operating system version needs to be at least Mac OS X 10.3 (Panther) in order to use the find panel. To purchase Mac OS X 10.3, or to learn more about it, visit Apple's Mac OS X website.

To use the find panel:

  1. Select Find… from the Edit > Find menu (or press Command-F).
  2. Type the text you want to find in the Find: text field, and click Next.
  3. If you want to replace something, enter what you want to replace in the Find: text box, type what you want to replace it with in the Replace with: text box, then click Next, and then click either Replace or Replace & Find. To replace all occurrences, click Replace All.

To find again after you've closed the find panel, you can press Command-G.
You can use the selected text as the text to find by selecting Use Selection for Find from the Edit > Find menu (or press Command-E).
To get rid of something throughout your document, enter what you want to get rid of in the Find: text field, leave the Replace with: text field blank, and then click Replace All.
