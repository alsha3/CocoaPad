Automatic Backup

A dilemma faced by all computer users is saving…and crashes, power outages, etc. Some programs (especially word processors) have come up with a way to prevent unsaved work from being lost. CocoaPad includes a method to prevent unsaved work from being lost as well: Automatic Backup.

Automatic Backup, if turned on in the Preferences panel, will backup all documents currently open in CocoaPad at specified intervals (5 minutes by default). Normally, when CocoaPad quits, it deletes the backups. But if a crash, blackout, or other kind of disaster prevents CocoaPad from quitting properly, CocoaPad won't be able to delete the backups. Every time CocoaPad starts up, it checks to see if the backups were deleted or not. If the backups still exist, then CocoaPad will recover the documents, and alert you that you need to re-save these documents.
CocoaPad will only backup documents that are open inside CocoaPad. CocoaPad is not a backup program, and it will not backup, say, your documents folder to a FireWire drive; nor will CocoaPad backup your open Microsoft Excel documents. It will only backup documents that you are working on in the CocoaPad program.

If you don't want CocoaPad to automatically backup your documents, you can turn this feature off in the Preferences panel. If you choose to enable this feature, then you can also change how often (in minutes) CocoaPad automatically backs up all open documents.
If you set the interval to 0 or less (e.g. "-5"), then you may experience erratic behavior, such as crazy slow typing, the cursor always moving to the end of a document, or not being able to select text. This is due to the fact that with an interval of 0 or less, CocoaPad is trying to backup your documents all the time. Hence, CocoaPad can get a bit confused about other things. We recommend you use a backup interval of 1 minute or longer.
In order for your changes to the backup interval to take effect, you can either press Return once you're done editing the text, switch to a different preferences tab, or close the window. Otherwise, the old backup interval will still be used until you do one of these three things.
